# Preep Docs — Магазин готовых юридических документов

## 🚀 О проекте
Preep Docs — это платформа, где можно купить готовые юридические документы. 
Никакой генерации, только проверенные шаблоны.

## 💰 Монетизация
- Документы продаются поштучно от 149 до 399 ₽
- Средний чек: 2-3 документа (500-700 ₽)
- Маржинальность: 90% (один раз создал — продаешь бесконечно)

## 📦 Структура
- `index.html` — главная страница
- `style.css` — все стили
- `script.js` — вся логика
- `api/` — бэкенд для платежей

## 🔧 Установка на Vercel
1. Загрузи все файлы на GitHub
2. Подключи репозиторий к Vercel
3. Добавь переменные окружения в Vercel
4. Готово!

## 💳 Платежная система
Для приема платежей нужно подключить:
- ЮKassa (Сбербанк) — для карт РФ
- ЮMoney — для кошельков
- СБП — для переводов по номеру телефона

## 📊 SEO
Каждый документ — это отдельная страница в поиске:
- "договор аренды скачать"
- "расписка о долге бланк"
- "претензия на возврат товара образец"
